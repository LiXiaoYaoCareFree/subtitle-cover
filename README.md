# Subtitle Cover 🎬

让字幕“消失”，不打断观影节奏。  
一个轻量、直观、可保存配置的字幕遮挡工具。✨

---

## 📌 产品简介

当视频自带字幕无法关闭、字幕语言不需要，或字幕位置影响画面时，`Subtitle Cover` 可在字幕区域上方创建可调节遮挡层，让你专注内容本身。

**你会得到：**
- 始终置顶：遮挡层始终位于最上方，不被其他窗口覆盖
- 实时可调：位置、尺寸、颜色、透明度随时调整
- 操作直觉：拖动即移动、手柄即缩放、右键即管理
- 配置持久：一次调好，后续启动直接复用

## 🛠️ 技术选型（开发语言）

本项目使用 **Swift** 开发，并按平台提供原生实现：
- macOS：`Swift + AppKit`
- Windows：`Swift + WinSDK (Win32 API)`

选择原因如下：
- 与 macOS 窗口系统深度集成，最适合实现“始终置顶、透明遮罩、右键菜单、触控板交互”
- 与 Windows 原生 Win32 窗口系统直接对接，易于实现置顶和透明窗口
- 对窗口层级、事件响应、颜色面板、快捷键支持更稳定
- 无跨平台运行时依赖，发布体积更小、性能更好

## ▶️ 本地运行（开发模式）

### 环境要求
- 已安装 Swift（5.9+）

先清理构建缓存：

```bash
swift run SubtitleCoverLauncher
```

```bash
swift build -c release
```

## 🪟 Windows 版本（Swift）

### 当前实现
- 可运行可拖动的置顶遮挡窗口
- 右下角灰色手柄缩放窗口

## 🚀 核心能力

### 1) 🧱 自由遮挡
- 拖动窗口到字幕区域即可覆盖
- 按需调整宽高，适配不同视频比例和字幕布局

### 2) 🌗 透明度微调
- 双击右下角灰色手柄，打开设置面板后调节透明度与颜色（黑色/白色）
- 支持范围：`10% ~ 100%`

### 3) 🎨 颜色自定义
- 内置颜色选择器
- 可根据画面风格选择更自然的遮挡色

### 4) 💾 配置保存
- 支持保存窗口位置、大小、颜色和透明度
- 下次启动可直接沿用

## ⚡ 快速上手（30 秒）

1. 启动程序，将遮挡器拖到字幕区域。  
2. 拖动右下角灰色手柄，调整到刚好覆盖字幕。  
3. 双击右下角灰色手柄，调节透明度并应用保存。  
4. 右键打开菜单，点击“保存设置”。  

## 🕹️ 操作说明（通用）

### 🪟 窗口控制
- 拖动：按住左键移动窗口
- 缩放：拖动右下角灰色手柄
- 设置：双击右下角灰色手柄，打开设置面板

### 🧭 右键菜单
| 菜单项 | 功能 |
| --- | --- |
| 更改颜色 | 打开颜色选择器 |
| 重置大小 | 恢复默认尺寸（`800 × 30`） |
| 窗口居中 | 在屏幕水平方向居中 |
| 保存设置 | 保存当前配置 |
| 帮助 | 查看快捷键与使用说明 |
| 退出 | 关闭程序 |

## 🍎 MacBook 专用交互

为适配 MacBook 触控板，推荐使用“一操作一功能”映射。

### ✅ 推荐映射
| 操作 | 功能 |
| --- | --- |
| `Control + 触控板拖拽` | 框选并创建新的遮盖区域 |
| `单指拖拽遮盖区` | 移动遮盖区位置 |
| `拖拽右下角手柄` | 调整遮盖区大小 |
| `双击右下角灰色手柄` | 打开设置面板（透明度） |
| `右键遮挡区` | 打开功能菜单（颜色、重置、居中、保存、帮助、退出） |

### ⚠️ 使用注意
- 仅在按住 `Control` 时进入框选模式，普通拖拽只用于移动，可减少误触
- `Control + 点击` 在 macOS 中常等同右键，建议使用 `Control + 拖拽` 完成框选
- 框选建议从字幕区域左上角拖到右下角，便于快速覆盖完整字幕

## 💡 使用建议

- 首次使用建议按“位置 -> 尺寸 -> 透明度/颜色”顺序调整
- 遮挡颜色建议贴近当前画面主色，视觉更自然
- 明亮画面可适度降低透明度，暗色画面可适度提高透明度
- 观看不同来源视频时，可随时拖动位置快速适配

## ❓ 常见问题

### 为什么建议保存设置？📝
保存后可复用当前参数，下次启动无需重复调节，尤其适合固定播放场景。

### 遮挡颜色怎么选更自然？🎞️
建议优先选择与视频底部区域接近的颜色，再通过透明度细调，避免突兀遮挡感。
